# Jarvis AI Assistant 🤖

Jarvis AI Assistant is a Python-based voice assistant inspired by Iron Man’s Jarvis. The assistant is designed to perform various everyday tasks using voice commands and artificial intelligence.

It can recognize speech, respond with voice output, provide weather updates, fetch news headlines, play music, manage reminders and schedules, and answer general queries using Gemini AI integration.

This project combines concepts of automation, APIs, speech recognition, file handling, and AI to create an interactive desktop assistant experience.

## Features

* Voice Recognition
* AI-Powered Responses
* Weather Updates
* News Headlines
* Music Playback
* Reminder System
* Task & Notes Management
* Scheduling System
* Morning Briefing

## Technologies Used

* Python
* SpeechRecognition
* Gemini API
* gTTS
* pygame
* WeatherAPI
* NewsAPI
